# NestJS Demo
A demo with **React** and **NestJS**.

## Features
- **react-hook-form** and **Zod** for form handling and validation.
- Business logic separated form the UI using custom hooks.
- Current user state persisted globally in **Zustand**.
- **NestJS** on top of **Express**.
- Zod-based global pipe with the centralised validation flow but more flexibility and control.
- JWT-based authentication.
- Both role-based and resource-based authorisation with **CASL**.

## Tech Stack
### Frontend
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Zustand

### Backend
- NestJS
